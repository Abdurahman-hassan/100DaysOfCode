# Coffee Machine

It's a funny program that simulates the Coffee Machine

## Example Input/OutPut

```
What would you like? (espresso/latte/cappuccino): latte

Please insert coins.
how many quarters?: 44
how many dimes?: 33
how many nickles?: 2
how many pennies?: 1

Here is $11.91 in change.
Here is your latte ☕️. Enjoy!

Sorry there is not enough water.

Water: 100ml
Milk: 50ml
Coffee: 76g

Please insert coins.
how many quarters?: 33
how many dimes?: 2

how many pennies?: 4
```
